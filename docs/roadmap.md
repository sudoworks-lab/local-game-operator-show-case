# Roadmap

## 次の拡張候補

- helper を Windows native addon 化しやすいよう、`HelperBridge` 契約を維持したまま backend を差し替える
- desktop capture の差分比較や frame sampling を追加して provider 入力を安定させる
- 複数 provider を切り替える UI と profile 管理を追加する

## 中期候補

- マウス操作の追加
- マクロ保存と再生
- Unity / Roblox 向けの target profile
- 連続実行ジョブの可視化

## 先送りにしたこと

- 配布向けインストーラ
- 自動更新
- クラウド同期
- 複数 PC 前提の設計
